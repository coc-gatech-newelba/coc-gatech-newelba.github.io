Project Tawny

A continuation of Project Elba.


Automated Large Scale Systems Experimentation. Declarative framework for creating, deploying and analyzing large-scale cloud-based experimental computer systems. Begin by clicking to start the Bootstrap Tutorial.

What is Project Tawny?

An automated experimental systems infrastructure to support identifying, isolating and analyzing performance anomalies occurring over millisecond-level timescales.

Easy to install and publish

Get started by cloning source into GitHub account of your project. Thanks to GitHub Pages, it will be automatically compiled and published under your project's subdomain on `github.io`.
